# AI-Powered Hybrid Speech Suite

## Overview

AI-Powered Hybrid Speech Suite is an intelligent speech processing platform that combines Machine Learning and Deep Learning techniques for audio analysis.

The system performs:

* Gender Classification from voice samples
* Speech-to-Text Transcription using OpenAI Whisper
* Intelligent Spell Correction using A* Search
* Audio Feature Extraction using MFCC
* Voice Pattern Discovery using K-Means Clustering
